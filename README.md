# Visa Tracker

A comprehensive visa application tracking system.

## Features
- Track visa application status
- Document management
- Timeline tracking
- Notifications and reminders

## Tech Stack
- Frontend: React/Next.js
- Backend: Node.js/Express or Spring Boot
- Database: PostgreSQL/MongoDB

## Getting Started
```bash
git clone git@github.com:udayganeshK/visa-tracker.git
cd visa-tracker
npm install
npm start
```
